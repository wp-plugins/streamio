== Description ==

Plugin that adds support for Streamio video hosting. Currently only supports adding oEmbed support which lets the
user paste http://s3m.io/ shorthand links and have the video/player automatically embed without having to bother
with special tags or embed codes.

Coming soon:

* Higher integration through the official API

== Installation ==

1. Search for 'Streamio' from the Plugins > Add New screen.
2. Install and activate the plugin.

== Screenshots ==

1. Embedded video in editor visual mode
2. Video link in editor text mode
3. Video displayed in published post on frontpage
